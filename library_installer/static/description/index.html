Library Installer

Install external dependencies require for third-party modules without logging in to your server

How to use it?

Install the module from Apps.

Click on "Install Python Library" which is located in apps menubar.

Type the libraries name with or without version name seperated by commas (,). Refer the screenshot for better understanding

New feature has been finalized. After new update you would be able to install by uploading requirements.txt file.
