Product Bundle Pack

The purpose of this module is to modify the packaging. When you define a new packaging, now you can add multiple product items in packaging (like cardboard boxes, filling material, etc). So when you use this packaging (eg. in a deliverly) the products you tied to packaging are added to the stock picking for the delivery.


Below listed are features of Product Bundle Pack Packaging: Product Bundle Pack offers you to add several products for sale as one pack product which you can sell as pack of multiple products.

Prerequisites before Installation

There are no prerequisites required before the Installation

Configurations

As shown in the image first activate the packaging by going to Sale settings. After activating go to the products, select the product or create a new product then go to the packaging add multiple products.

After Creating the packaging you can see your package.

Now go to Sale Order and select the package product now in Package field drop down you can see your package created.

After confirming the sale order click on delivery button and now you can see all of your products in delivery which you have selected in Pack.

After validating it. Go to inventory -> Transfer You can see that your pack is successfully created and delivered in pack.
