Track Stock Move for Unit/Department (Quota)

By using this module you will be able to:

  • 1) Track stock for unit/service
  • 2) Affect a percent of quantity by unit during a move
  • 3) Limit products quantity using by unit/service during a period
  • 4) See what a unit/service has consume in the unit form view
